Description
This is an homage to Planetarium by Daniel Linssen. Go check out his page if you haven't yet.

This is not a game. Stellarium is a procedural solar system generator using real Newtonian physics. If you would like to develop something with this physics system please check out Gravity Simulator on my page. it contains all the source code and documentation for you to get started.
