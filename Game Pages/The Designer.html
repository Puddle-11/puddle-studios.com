Description
In this serene moonlit landscape, instead of playing as the player, you play the level/level designer. creating the level, guiding the player to completion. the player is totally autonomous. the user has no control over his movements.
Disclaimer
this game was supposed to be a submission for the 2023 GMTK game jam. i plan to make minor QOL updates over the following week, but do not plan to turn this into a full game, if you would like that, let me know.
